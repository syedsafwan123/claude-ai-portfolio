## ✨ Features

- 🌌 **Animated particle network** with connected dots across the hero background
- 💬 **Live AI chat mockup** that cycles through responses with a typing effect
- 📊 **LLM Comparison Table** — Claude vs GPT-4o vs Gemini Ultra vs Llama 3
- 🔄 **Billing toggle** — switch between Monthly and Annual pricing with animated counters
- 🎨 **3D card tilt effect** on hover across all cards
- 📈 **Scroll progress bar** at the top of the page
- 🖱️ **Cursor glow** effect following the mouse
- 🎞️ **Reveal animations** triggered as you scroll into view
- 💥 **Ripple effect** on all buttons
- 🌗 **Dark mode first** — premium dark aesthetic throughout
- 📱 **Fully responsive** — works on mobile, tablet, and desktop

## 🚀 Running Locally

### Option 1 — Python (no install needed)
```bash
cd "claude-ai-portfolio"
python -m http.server 5500
```
Then open: **http://localhost:5500**

### Option 2 — VS Code Live Server
1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
2. Right-click `index.html` → **Open with Live Server**

### Option 3 — npx serve
```bash
npx serve .
```

## 🧠 Sections Overview

### Hero
Full-screen landing with a floating Claude chat mockup that demonstrates live AI conversations, animated particle stars, and key stats (200K context, 100+ languages, 99% safety).

### About Claude
Three cards highlighting Constitutional AI, Human-Centered Design, and Versatile & Scalable architecture.

### Capabilities
Six feature cards showcasing Advanced Reasoning, Code Generation, Creative Writing, Research & Synthesis, Multilingual Support, and Vision & Multimodal capabilities.

### Claude vs The Rest
An interactive comparison table pitting Claude against GPT-4o, Gemini Ultra, and Llama 3 across 7 benchmark metrics — with animated progress bars and a highlighted "Winner" Claude column. Four summary cards reinforce Claude's top positions.

### Pricing Plans
Four-tier pricing (Free, Pro $20/mo, Team $30/mo, Enterprise Custom) with a live billing toggle for monthly vs annual pricing.

### Real-World Integrations
- 💻 **Technology** — Code window with live Python API snippet
- 🏥 **Healthcare** — Medical dashboard with drug interaction alerts & AI insights
- 🚗 **Automotive** — Interactive car dashboard with speedometer & AI copilot

### Testimonials
Reviews from a CTO, Chief Medical Officer, and VP of Innovation.
